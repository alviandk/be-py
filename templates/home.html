Kenapa Pemrograman Python?

Pemrograman dengan python memiliki beberapa kelebihan: syntax mudah dibaca, popularitas tinggi, implementasi di banyak bidang dan eksistensi.

source: Code School

Syntax Mudah Dibaca

Seringkali programmer lupa dengan syntax coding yang sudah diimplementasi dan harus membaca ulang programnya. Dengan syntax yang mudah dibaca, memudahkan programmer untuk memahami kembali coding yang pernah ditulis. Selain itu, syntax mudah dibaca menjadikan proses pembelajaran bahasa python lebih mudah. Oleh karena itu, syntax python lebih bersahabat untuk orang yang baru belajar pemrograman.

source: Stack Overflow

Popularitas Tinggi

Populer artinya banyak orang yang menggunakan. Saat menemukan error atau ingin memecahkan masalah dengan python, jawaban googling yang ditemukan akan lebih mudah didapatkan. Karena kemungkinan sudah ada orang lain yang pernah menemukan masalah yang sama dan bisa memecahkannya. Hal ini akan meningkatkan produktivitas programmer dalam mengembangkan aplikasi.

source: Hackbright Academy

Implementasi di Banyak Bidang

    Python bisa digunakan untuk mengembangkan:
  • aplikasi berbasis web dan desktop
  • data science
  • machine learning
  • image recognition
  • automated test
  • aplikasi geo location dan lain lain
source: Hackbright Academy

Eksistensi

Pemrograman python banyak digunakan oleh perusahaan besar seprti google, dropbox, instagram dan masih banyak lagi. Selain itu, banyak framework favorit programmer yang dibuat dengan python. Seperti: django (aplikasi web), flask (aplikasi web), odoo (aplikasi erp), numpy (aplikasi data science), pandas (aplikasi data science), tensorflow (aplikasi machine learning) dan masih banyak lagi yang lainnya.
