Tentang Kami

Be-Py merupakan singkatan dari Belajar Python. Be-Py terbentuk karena banyak mahasiswa jurusan IT / Ilmu Komputer yang merasa belum memiliki skill programming, sedangkan skill programming merupakan skill wajib yang harus dimiliki oleh mahasiswa jurusan IT / Ilmu Komputer.

Be-Py merupakan platform pembelajaran pengembangan perangkat lunak menggunakan bahasa pemrograman python. Materi pembelajaran akan ditulis dalam bahasa Indonesia. Materi pembelajaran terdiri dari dasar bahasa pemrograman python, membuat full stack website dengan django, membuat api dengan django rest framework, membuat microservice dengan flask dan lain lain.

Mengapa Python?

  • Keunggulan Python dibandingkan dengan bahasa pemrograman lain.
  • Keunggulan kemudahan dalam sintaks, enak dibaca, sehingga lebih mudah digunakan untuk berkolaborasi.
  • Rekomendasi ahli untuk belajar pemrograman bagi pemula.
  • Kebutuhan programmer.
  • Banyak web ternama yang menggunakan Python seperti: Google, Paypal, Youtube
